# Data Flow Diagram

```mermaid
sequenceDiagram
    participant U as User
    participant App as Mobile App
    participant HC as Health Connect
    participant API as Backend API
    participant AI as AI Service
    participant DB as Database

    U->>App: Open App & Login
    App->>API: Authenticate
    API->>DB: Verify Credentials
    DB-->>API: User Data
    API-->>App: Auth Token

    U->>App: Request Vitals
    App->>HC: Request Health Data
    HC-->>App: Health Metrics
    App->>App: Process & Display

    U->>App: Upload Image for Analysis
    App->>AI: Send Image
    AI-->>App: Analysis Result
    App->>DB: Store Result
    App-->>U: Display Result

    U->>App: Book Appointment
    App->>API: Appointment Request
    API->>DB: Store Appointment
    DB-->>API: Confirmation
    API-->>App: Booking Confirmed
    App-->>U: Show Confirmation

    U->>App: Store Document in Vault
    App->>App: Encrypt Document
    App->>DB: Store Encrypted
    DB-->>App: Storage Confirmed
    App-->>U: Document Saved
```
